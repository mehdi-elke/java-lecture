### Plan du cours Java pour Débutants

#### 1. Introduction à Java
- Historique et caractéristiques du langage
- Installation du JDK
- **Compilation et exécution en ligne de commande** : 
  - Utilisation de `javac` pour compiler une classe
  - Utilisation de `java` pour exécuter une classe
- Premier programme Java ("Hello World")
- Configuration de l’environnement (IDE)

#### 2. Syntaxe de base et concepts fondamentaux
- **Types de données primitifs** : int, double, char, boolean
- **Variables et constantes** : déclaration, initialisation, conventions de nommage
- **Chaînes de caractères (String)** : méthodes (`length()`, `charAt()`, `substring()`, `indexOf()`, `equals()`, `toUpperCase()`, `toLowerCase()`), `StringBuilder`

#### 3. Les opérateurs en Java
- Opérateurs arithmétiques : `+`, `-`, `*`, `/`, `%`
- Opérateurs relationnels : `==`, `!=`, `>`, `<`, `>=`, `<=`
- Opérateurs logiques : `&&`, `||`, `!`
- Opérateurs d’incrémentation/décrémentation : `++`, `--`
- Opérateurs d’affectation : `=`, `+=`, `-=`, `*=`, `/=`, `%=`

#### 4. Structures de contrôle
- Conditions : if, if-else, switch
- Boucles : for, while, do-while
- Utilisation de `break`, `continue`
- Switch avec expressions (Java 12+)

#### 5. Les tests unitaires
- Junit
- Mockito

#### 6. Structures de données, collections, interfaces fonctionnelles et lambdas
- Collections (Listes, Set, Map) : manipulation, ajout, suppression, itération
- **Interfaces fonctionnelles** : `Function`, `Predicate`, `Consumer`, `Supplier`
- **Lambdas** : syntaxe, utilisation des expressions lambda, utilisation avec les interfaces fonctionnelles

#### 7. POO
- Comprendre la différence entre type objet et type référence
- Polymorphisme : concept et utilisation
- Programmation orientée objet (POO) : classes, objets, héritage, interfaces
- Encapsulation, héritage, polymorphisme
- Casting des objets et types primitifs
- Utilisation de l'opérateur `instanceof`
- Pattern Matching avec `instanceof`

#### 8. Gestion des exceptions
- Introduction aux exceptions
- Try-catch-finally
- Création d'exceptions personnalisées

#### 9. Gestion des dates en Java
- Utilisation des classes `LocalDate`, `LocalTime`, `LocalDateTime`
- Manipulation des dates : ajout/soustraction de jours, mois, années
- Formatage des dates avec `DateTimeFormatter`
- Gestion des fuseaux horaires avec `ZonedDateTime`

#### 10. Génériques en Java
- Introduction aux génériques
- Classes et méthodes génériques
- Borne supérieure et inférieure des types génériques

#### 11. Fichiers et entrées/sorties (I/O)
- Lecture et écriture dans des fichiers (BufferedReader, BufferedWriter)
- Utilisation des classes File et Files
- Sérialisation des objets
- Gestion des flux (InputStream, OutputStream)

#### 12. Concurrence
- Introduction aux threads
- Synchronisation des threads

#### 13. Connexion à une base de données
- JDBC : Concepts et utilisation
- Connexion à une base de données MySQL
- Requêtes SQL depuis Java (SELECT, INSERT, UPDATE, DELETE)

#### 14. Interfaces utilisateur (Swing)
- Introduction à Swing (JFrame, JPanel, JButton, JTextField)
- Layouts (BorderLayout, GridLayout)
- Gestion des événements (ActionListener, EventHandler)
